# RTA-Fare (Dubai Metro)

A simple iOS app that calculates Dubai Metro fares based on the number of zones traveled and the selected Nol card type.

Built using Swift and SwiftUI as a practice project to learn iOS app development.

⸻

**Features**

- Select number of zones traveled
- Choose Nol card type
- Instantly calculates the correct metro fare
- Simple and clean SwiftUI interface
- Fast and lightweight

⸻

**Technologies Used**

- Swift
- SwiftUI
- Xcode

⸻

**Fare Logic**

The app calculates fares based on the official Dubai Metro zone system.

⸻

**Purpose**

This project was built to:

- Practice iOS app development
- Learn SwiftUI layout and state management
- Implement real-world logic in an app
- Build projects for my developer portfolio

⸻

**Future Improvements**

Possible upgrades for the app:

- Add Dubai Metro station selection
- Automatic zone detection between stations
- Better UI design
- Add bus and tram fare calculations
